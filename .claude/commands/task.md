Взять задачу в работу.

## Шаг 1: Контекст
Прочитай `.claude/CLAUDE.md`.

## Шаг 2: Доступные задачи
```bash
ls Tasks/backlog/ | grep .md
```

### Если задача указана (`/task T001`) — сразу к Шагу 3.
### Если задач несколько — покажи список и жди выбора.
### Если задач нет — сообщи.

## Шаг 3: Выполни задачу

1. Прочитай файл задачи полностью
2. Выведи план
3. Выполняй по шагам
4. После каждого шага — тесты через MCP:
   ```bash
   python scripts/mcp_call.py run_shell_command '{"command": "cd /home/flomaster/teatr-bot && python3 -m pytest tests/ -v --tb=short 2>&1 | tail -30"}'
   ```
5. По завершении:
   - Обнови статус задачи на DONE
   - Залей код на сервер через MCP
   - Перезапусти бота

## Правила
- Сервер — через MCP, не SSH
- Claude API вызывать только из brain/digest_builder.py
- Комментарии — на русском
- Тесты с ошибками — фиксировать, не скрывать
- Не запрашивать KudaGo чаще 1 раза в 6 часов
