# /fin_context — Фиксация контекста без деплоя

Только сохранение контекста в память. Без git, без сервера, без тестов.

## Шаг 1: Сохранение контекста в память
Обнови или создай файл `memory/project_session_log.md (в директории auto memory проекта)` с:
- Дата сессии
- Что было сделано в этой сессии (кратко)
- Текущий статус задач (T001-T004)
- Открытые проблемы / блокеры
- Что делать в следующей сессии

## Шаг 2: Обновить CLAUDE.md
Обнови статусы задач в `.claude/CLAUDE.md` если они изменились за сессию.

## Шаг 3: Краткий отчёт
```
══════════════════════════════
  Контекст сохранён — $DATE
══════════════════════════════
Задачи:    T001 ✅ T002 ✅ T003 ⬜ T004 ⬜
Сделано:   [что сделано за сессию]
Следующее: [что делать дальше]
══════════════════════════════
```
