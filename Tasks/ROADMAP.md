# Театральный бот — Roadmap задач

> Создано: 14 марта 2026
> T001 ⬜ | T002 ⬜ | T003 ⬜ | T004 ⬜

---

## ⬜ В очереди

**T001 — БД + KudaGo-коллектор**
PostgreSQL схема (theaters, shows, show_dates, rss_news, bot_events), Docker-контейнер на порту 5435, коллектор KudaGo API с пагинацией и кэшированием в БД, скрипт ручного запуска сбора данных.

**T002 — Telegram-бот + дайджест**
Базовый бот с командами /digest /today /weekend /week /status. Inline-кнопки для выбора периода. Claude-powered дайджест через digest_builder.py. Graceful degradation — raw список если Claude недоступен.

**T003 — RSS + обогащение контекста**
RSS-коллектор для 5+ крупных театров (Большой, МХТ, Вахтангова, Ленком, Современник). Связь новостей RSS с записями в shows. Обогащение дайджеста контекстом (премьеры, гастроли, награды). Команда /premieres.

**T004 — Тесты + деплой**
Unit-тесты для коллекторов и digest_builder (mock API). Интеграционные тесты с реальной БД. Деплой на сервер через MCP. Ежедневный планировщик в production.
